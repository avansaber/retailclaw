# RetailClaw

Retail management for [ERPClaw](https://github.com/avansaber/erpclaw). Pricing and promotions, loyalty programs, visual merchandising, wholesale/B2B operations, and returns and exchanges. 57 actions across 6 domains.

## Features

- **Pricing** -- price lists, promotions (percentage, fixed, BOGO), coupon management, activation workflow
- **Loyalty** -- loyalty programs, member enrollment, points earning and redemption, tier management, gift cards
- **Merchandising** -- product categories, planograms, shelf positioning, display management
- **Wholesale/B2B** -- wholesale customers, tiered pricing, bulk orders, credit limits, payment terms
- **Returns & Exchanges** -- return authorizations, return items, processing, exchanges, restocking fees, store credit
- **Reports** -- channel performance, margin analysis, loyalty analytics, category performance, promotion effectiveness, inventory turnover

All financial transactions use ERPClaw's double-entry General Ledger with Decimal precision.

## Installation

Requires [ERPClaw](https://github.com/avansaber/erpclaw) core.

```
install-module retailclaw
```

## Quick Start

```
"I'm opening a retail store -- set me up"
```

The bot will install RetailClaw and guide you through setting up price lists, loyalty programs, and merchandising.
